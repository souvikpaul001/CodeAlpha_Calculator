🧮 Simple Calculator

A basic and responsive calculator built using HTML, CSS, and JavaScript.
It performs standard arithmetic operations such as addition, subtraction, multiplication, and division. Perfect for beginners learning DOM manipulation and event handling in JavaScript.

🚀 Features

• Clean and simple UI

• Responsive design

• Supports basic arithmetic operations

• Interactive button clicks

• Error handling (e.g., division by zero)

• Lightweight and beginner-friendly code

🛠️ Technologies Used

• HTML5 – structure of the calculator

• CSS3 – styles, layout, and responsiveness

• JavaScript (Vanilla JS) – calculator logic and interactivity
